# STRATCOM.INTEL - Version Checkpoint: "safe"

This document serves as a logical snapshot of the application state as of this moment.

## Core Modules
- **Head-to-Head Analysis**: Comparative matrix with dual-country selection and randomized force generation.
- **Coalition Simulator**: Aggregate force projection engine with expanded tactical metrics (Fighter Jets, Submarines, Nuclear Warheads).
- **Global Strength Map**: Geospatial distribution visualizer with sidebar-integrated Target Profile intelligence.
- **Historical Trajectory**: Longitudinal analysis of expenditure and personnel scaling (1995-2023).

## UI/UX Framework
- **Theme**: High-contrast Tactical Dark Mode with Amber/Gold accents.
- **Components**: Sidebar navigation, STRATCOM-branded cards, custom command-interface country selectors.
- **Intelligence Overlay**: Unified noise-filter texture and tactical instructions.

## Data Layer
- **Unified Scoring Engine**: Normalized 0-100 index based on manpower, air, ground, naval, economy, and logistics.
- **Intelligence Sources**: IISS, SIPRI, GFP, and FAS integrated via Tooltip telemetry.
